1. MyLocalTon
2. Multiplatform wrapper of TON validator. Java GUI application allows user to run up to 7 validators locally on his PC. Contains blocks', transactions' and messages' explorer with the search engine. Creation of various wallets, ton nodes with a single click. Visual representation of TON elections.
3. Well tested on multiple platforms, proved by time and TON community.
